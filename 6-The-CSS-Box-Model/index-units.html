EM's

Are relative units with font-size, 1em equals the font-size of the parent. 2em's is twice the font-size of the parent, etc. With other properties, 1em is equal to the computed font-size of the element itself

REM's

Relative to the root html element's font-size. Often easier to work with. If the root font-size is 20px, 1 rem is always 20px, 2rem is always 40px, etc.
